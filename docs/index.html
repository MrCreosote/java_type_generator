Java Type Generator

Program generates java client and server classes for JSON RPC calls.
Usage: <program> [options...] <spec-file>
Usage: <program> {-h|--help}     - to see this help
 <spec-file>  : File *.spec for compilation into java classes
 -S           : Defines whether or not java code for server side should be created,
                default value is false, use -S for true
 -g <gwtpckg> : Gwt client java package (define it in case you need copies of
                generated classes for GWT client)
 -l <lib-dir> : Library output folder (exclusive with -o, not required when using -s)
 -o <out-dir> : Output folder (src and lib subfolders will be created), use -s and
                possibly -l instead of -o for more detailed settings
 -p <package> : Java package parent (module subpackages are created in this
                package), default value is us.kbase
 -s <src-dir> : Source output folder (exclusive with -o)
 -t <tmp-dir> : Temporary folder, default value is parent folder of <spec-file>
 -u <url>     : Default url for service
